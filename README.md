# PostBox.app

This application will be used to monitor postbox. It has simple functionalities for users convenience.

Notifications about:
  - New messages.
  - Who send the letter.
  - About overload (when letter limit is almost full).
  - And even about brake in.

User's functionalities:
  - User can see letters history, who and when sent to letter.
  - User can indicate letters for delete without read.
